Very basic example of server actions in Next JS. Just for my exercise.

## Before Trying Server Actions

- First, you need to add experimental option `serverActions: true` in your next config

```js
module.exports = {
  experimental: {
    serverActions: true,
  },
}
```

- Enjoy
